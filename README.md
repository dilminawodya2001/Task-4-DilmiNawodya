## 📌 Overview

This Power BI dashboard provides an end-to-end visual analysis of e-commerce business performance. Built as part of the **DecodeLabs Industrial Training Program (Batch 2026)**, this project demonstrates real-world data visualization skills using Power BI Desktop, connected to the `project_4` database.

The dashboard enables stakeholders to monitor key business metrics, identify sales trends, understand customer behavior, and make data-driven decisions — all through interactive and drill-through visuals.

## 📊 Dashboard Features

The dashboard covers the following analytical areas:

- **Sales Overview** — Total revenue, order volume, and average order value (AOV) over time
- **Product Performance** — Top-selling products, category-wise breakdown, and revenue contribution
- **Customer Analysis** — New vs. returning customers, customer segmentation, and regional distribution
- **Order & Fulfillment Trends** — Order status tracking, delivery performance, and return rates
- **Time-Series Analysis** — Monthly, quarterly, and yearly comparisons with trend lines
- **KPI Cards** — At-a-glance metrics for quick executive summary

## 🚀 Getting Started

### Prerequisites

- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed (latest version recommended)
- Access to the `project_4` database with valid credentials

### Steps to Open

1. **Clone this repository**
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```

2. **Open the report file**
   - Launch Power BI Desktop
   - Open `E-Commerce_Analysis_Dashboard.pbix`

3. **Configure the data source**
   - Go to **Home → Transform Data → Data Source Settings**
   - Update the server/host and credentials to point to your `project_4` database

4. **Refresh the data**
   - Click **Home → Refresh** to load the latest data
